# secret_santa

A simple auto emailer<br/>

Commands:
- CMD: list commands
- CRT: create a group
- ADD: add to group
- RM: remove from group
- WSPR: send message to secret santa
- ROLL: vote to reroll the pairings (NOT FEATURED)
- BCST: send message to all in group
